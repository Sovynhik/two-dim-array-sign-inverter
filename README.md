# TwoDimArraySignInverter — Лабораторная работа по двумерным массивам

**Java-приложение** для **инверсии знака элемента** в каждой строке **двумерного массива** с **минимальной разницей** между суммами слева и справа.

---

## Задача

> **Дан двумерный массив** с разной длиной строк.  
> Заполнить его **случайными числами** от `10` до `50`.  
>  
> Для **каждой строки** найти **элемент**, у которого **разница между суммой слева и суммой справа — минимальна**.  
> → **Инвертировать его знак** (`x → -x`).

---

## Пример работы

```text
Оригинальные границы: [10, 50]
Исходный массив:
[12, 45, 23, 18, 37]
[28, 15, 41, 33, 19, 46]
...

После инверсии:
[12, 45, -23, 18, 37]   ← 23 → -23 (min diff)
[28, 15, 41, -33, 19, 46] ← 33 → -33
...
Сумма первого и последнего: 58
```

## Как запустить

1. Убедитесь, что установлена **jdk1.8.0_144+**
2. Склонируйте репозиторий:
   ```bash
   git clone https://github.com/yourname/two-dim-array-sign-inverter.git
   ```
3. Перейдите в папку проекта и выполните:
   ```bash
   javac -d out -cp . src/ru/rsreu/savushkin0417/*.java src/ru/rsreu/savushkin0417/twodimarray/*.java
   java -cp out ru.rsreu.savushkin0417.ClientRunner
   ```
